
- If we want to open the default browser when we excute the start command, we can do the below:
```json
{
    "scripts": {
        "start": "ng serve -o"
    }
}
```

- Difference between `for...of` and `for...in`:
    - `for...of` iterates over iterable objects and gives us the value in array
    - `for...in` iterates over the properties of an object in the array and gives us the index (**Tip**: in like index)
